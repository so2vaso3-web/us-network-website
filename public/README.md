# Public Assets

Thư mục này chứa các file tĩnh (static files) cho website.

## OG Image (Open Graph Image)

File `og-image.png` (1200x630px) sẽ được sử dụng khi chia sẻ link trên:
- Facebook
- Twitter
- LinkedIn
- WhatsApp
- Và các mạng xã hội khác

### Cách tạo OG Image:

1. **Dùng file HTML có sẵn:**
   - Mở `og-image.html` trong trình duyệt
   - Chụp màn hình phần preview (1200x630px)
   - Lưu với tên `og-image.png`

2. **Dùng công cụ online:**
   - [Canva](https://www.canva.com) - Tạo ảnh 1200x630px
   - [Figma](https://www.figma.com) - Thiết kế chuyên nghiệp
   - [OG Image Generator](https://og-image.vercel.app) - Tạo tự động

3. **Yêu cầu:**
   - Kích thước: 1200x630px (tỷ lệ 1.91:1)
   - Format: PNG hoặc JPG
   - Kích thước file: < 1MB (khuyến nghị)
   - Nội dung: Logo + Tiêu đề + Mô tả ngắn
